# Energy Forecasting Project

Transformer-based model to forecast hourly electricity demand using PJM hourly energy consumption data.

## Folder structure
- data/: raw dataset
- notebooks/: exploration and preprocessing
- src/: reusable code
- figures/: plots for report/slides

## Dataset
PJM Hourly Energy Consumption dataset from Kaggle, using the PJME region.

## Current status
- Data preprocessing completed
- LSTM baseline implemented
- Transformer pipeline in progress
